# Todo list on React and Redux
## requirements
- node js (npm install) version >= 16

## setup
```bash
git clone https://github.com/maamri95/todo-list.git
cd todo-list && npm install
npm run dev
```
## usage
- add new todo
- delete todo
- edit todo
- mark todo as done
- mark todo as undone
- see all todos
- see cat in todo detail
